# Plantillas rápidas — Base funcional

Esta actualización fue creada exclusivamente sobre el ZIP funcional confirmado por el usuario.

## Funciones añadidas
- Biblioteca de plantillas por industria.
- Crear, editar, duplicar y eliminar plantillas.
- Partidas ilimitadas por plantilla.
- Categorías, favoritas, búsqueda y contador de uso.
- Tiempo estimado, garantía y notas internas.
- Importación y exportación JSON.
- Conserva el botón para duplicar el último servicio.

## Archivos modificados
- app.js
- styles.css

No se modificaron index.html, firebase-config.js, sw.js, firestore.rules, admin.js, mobile.js, portal.js ni el código de autenticación.
