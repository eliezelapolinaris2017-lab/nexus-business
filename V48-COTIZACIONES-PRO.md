# Nexus Business PR — V48 Cotizaciones Pro

## Incluido

- Nuevo módulo **Cotizaciones Pro** en el menú principal.
- Registro de cotizaciones con cliente, activo, equipo, fecha, vigencia, prioridad y estado.
- Partidas múltiples para servicios, materiales, mano de obra y extras.
- IVU automático usando el porcentaje configurado en Configuración.
- Estados: Borrador, Enviada, Aprobada, Rechazada y Convertida.
- PDF/preview corporativo premium para cotizaciones.
- Conversión directa de cotización a servicio.
- Report Center actualizado con reporte de cotizaciones.
- Búsqueda global y actividad reciente integradas.

## Flujo recomendado

Cliente → Cotización → Aprobación → Servicio → Factura → Cobro → Flujo de caja.
