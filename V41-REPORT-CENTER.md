# v41 Report Center

Cambios incluidos:

- Reportes organizados en una sola tarjeta limpia.
- Categorías: Operacionales, Financieros, Administrativos y Ejecutivos.
- Buscador de reportes.
- Scroll interno para evitar ruido visual.
- Acciones estándar por reporte: Ver, PDF y Exportar.
- Exportación CSV básica por tipo de reporte.
- Diseño responsive para iPhone, iPad y escritorio.

Notas:

- La lógica de vista previa y PDF existente se conserva.
- No se cambió la estructura de datos.
- No se añadieron módulos nuevos.
