# Editar precargado

La ventana de editar ahora carga todos los campos base del módulo y conserva los valores existentes del registro.

Incluye normalización de relaciones:
- Cliente
- Activo
- Empleado / Equipo
- Suplidor
- Compra
- Factura

Si un dato existe como nombre pero no como ID, el sistema intenta enlazarlo automáticamente para que el formulario no abra vacío.
