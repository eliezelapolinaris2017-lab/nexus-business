# v38 Services Edit Fix

Corrección aplicada:

- El módulo Services / Servicios ahora usa un editor completo propio.
- Al editar un servicio, el formulario se precarga con los datos existentes.
- Mantiene cliente, activo, equipo, fecha, estado, prioridad, tipo de servicio, descripción, monto, partidas, campos por industria y ruta de transporte.
- Las partidas se editan visualmente, no como JSON.
- En Transporte se precargan origen, destino, millas, tarifa, cargo base y botón de ruta.
- Al guardar actualiza el registro existente, no crea duplicado.
