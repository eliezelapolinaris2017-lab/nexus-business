Nexus Business PR V1

Base SaaS multi-servicios basada en Nexus Transport PR.

Incluye:
- Auth por usuario con Email/Password
- Datos privados por UID en Firestore
- Clientes
- Servicios / órdenes de trabajo
- Personal con comisiones y retenciones
- Suplidores / proveedores
- Recursos / flota / equipos
- Facturación
- Cobros
- Flujo de caja
- Reportes PDF corporativos
- Backup/importación

Firebase:
1. Activar Authentication > Email/Password.
2. Copiar firestore.rules en Firestore Rules.
3. Subir estos archivos a GitHub Pages.

Colección usada:
users/{uid}/nexusBusiness/main
