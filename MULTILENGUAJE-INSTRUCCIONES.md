# Nexus Multilenguaje

## Ubicación
Configuración → Idioma

Opciones disponibles:
- Español
- English

## Qué cambia
- Menú principal
- Dashboard / Centro de Control
- Formularios
- Estados
- Planes
- Reportes
- Facturas PDF / previews
- Textos operativos principales

## Nota técnica
El idioma se guarda en el perfil del usuario como `language`.
Valores:
- `es`
- `en`

El cambio aplica al guardar configuración y refrescar la vista.
