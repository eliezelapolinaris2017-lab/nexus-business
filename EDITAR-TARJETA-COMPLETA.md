# Corrección: Editar en tarjeta completa

Se reemplazó el `prompt()` simple por una ventana completa de edición.

## Ahora permite

- Abrir una tarjeta/modal completo.
- Editar todos los campos principales del registro.
- Editar clientes, servicios, activos, equipo, suplidores, compras, facturas, cobros, nómina y flujo de caja.
- Usar selectores para cliente, activo, empleado, suplidor, compra y factura.
- Guardar relaciones actualizando nombres asociados.

## Nota

Los campos complejos como `items`, `fields` y `route` se muestran en JSON para no perder datos avanzados.
