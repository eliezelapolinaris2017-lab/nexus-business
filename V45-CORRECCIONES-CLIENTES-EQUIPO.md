# v45 Correcciones Clientes y Equipo

## Clientes
- Las etiquetas ahora se seleccionan desde una lista con scroll vertical.
- Se evita escribir etiquetas manualmente y se reducen errores.
- Permite selección múltiple.

## Equipo
Se añadieron campos administrativos:
- Identificación personal ID.
- Licencia de conducir.
- Vehículo asignado, si aplica.

El vehículo asignado se alimenta desde los activos registrados como vehículo.
