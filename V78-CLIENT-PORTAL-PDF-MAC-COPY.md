# V78 — Portal PDF + copia en Mac
- Facturas y cotizaciones ahora incluyen botones Ver PDF y Descargar PDF.
- El portal publica partidas, subtotal, impuesto, pagos, balance, notas y términos.
- Copia del enlace compatible con Safari/macOS mediante Clipboard API y respaldo execCommand.
- Si Safari bloquea ambos métodos, muestra el enlace listo para copiar manualmente.
